This system was developed to automate a previously manual and time-consuming process that required multiple operators. It loops through a predefined list, inserting data into a web form. For each entry, the page returns client information, which is displayed in a DataFrame on the screen. The automation captures these results, stores them in memory until the list is complete, and then generates a unified Excel file for the user.

Before automation, this task was slow and required five operators to handle the daily demand. With this solution, the operation now requires only one operator for a few minutes to start the process, freeing time for other activities and significantly reducing costs.
